# Auralink 🌌

Auralink is a web-based emotional expression platform built during a local hackathon.  
It allows users to share emotions **without using words**, using visuals, cards, drawings, and interaction-based matching.

---

## 🚀 How to Run This Project (Local Setup)

Follow these steps to run **Auralink** on your local machine using **XAMPP**:

1. **Download the Repository**
   - Click on **Code → Download ZIP** from this GitHub repository.
   - Extract (unzip) the downloaded ZIP file.

2. **Move Project to `htdocs`**
   - Copy the extracted project folder.
   - Paste it into:
     ```
     C:\xampp\htdocs\
     ```

3. **Start XAMPP**
   - Open **XAMPP Control Panel**.
   - Start **Apache**.
   - Start **MySQL** (and import the given db.sql file).

4. **Run the Project in Browser**
   - Open any browser (Chrome, Edge, Firefox).
   - Go to:
     ```
     http://localhost/auralink
     ```
     

✅ The project should now run successfully.

## 🧠 Project Idea

**Auralink** is designed as a safe and creative digital space where users can express emotions **without text or verbal communication**.

Instead of words, emotions are shared through:
- Visual cards
- Drawings
- Interactive matching

This removes language barriers and encourages honest emotional expression.

---

## ✨ Core Features

### 1️⃣ Mood Post Wall
- A shared wall where users post their current emotional state.
- No text-based posts.
- Emotion-first interaction.

### 2️⃣ Emotion Card Posts
- Users choose cards that represent their mood.
- Each card visually symbolizes a feeling or mental state.
- Simple and expressive.

### 3️⃣ Draw Canvas
- A free drawing canvas.
- Users can draw anything on their mind.
- Encourages abstract and raw expression.

### 4️⃣ SyncBoard (Emotion Matching)
- Card-based matching system.
- Connects users with similar emotional states.
- Helps users find emotional companions or friends.
